## Running Locally

Clone the project

```bash
https://github.com/TheDanniCraft/TerminalWebsite.git
```

Go to the project directory

```bash
cd TerminalWebsite
```

Remove remote origin

```bash
git remote remove origin
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```
